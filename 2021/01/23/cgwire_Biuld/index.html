2.添加新用户帐号

添加新账号后，点击发送邀请验证会失败，可以不用管。

默认密码是 default

登录帐号邮箱可以是个不存在的邮箱，自定义

第一次登录是输入默认密码（ default ) 就能登录，第一次登录会提示修改密码。（要记住密码，忘记后邮箱无法找回）
